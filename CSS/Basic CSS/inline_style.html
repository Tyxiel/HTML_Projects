CSS3 Inline Style

Este estilo é feito dentro do própio elemento

Exemplo:

Código:

<p style="color: green;">

Olá, eu sou Tyxiel!

</p>

Resultado:

Olá, eu sou Tyxiel

Exemplo 2:

Código

<span style="background-color: rgba(0, 0, 150, 0.2)">

Bem vindo ao meu GitHub!

</span>

Resultado:

Bem vindo ao meu GitHub!

Isso retira a necessidade de abrir um estilo externo, além de ser mais prático
